Reachy Mini App

BoBe

A Claude-backed voice assistant for Reachy Mini. Say “Hey Jarvis” to wake.

Getting Started

Run locally with uv

Install the app dependencies, start the daemon, then launch BoBe.

uv sync --group dev
uv run reachy-mini-daemon --sim  # in another terminal
uv run bobe --gradio

Configuration

Voice bridge + Claude answers

Set OPENAI_API_KEY for the realtime speech bridge and ANTHROPIC_API_KEY for Claude-backed BoBe answers.
